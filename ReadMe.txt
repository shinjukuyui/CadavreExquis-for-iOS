HyperCardスタック「優美なる死体 v2.1」のiOS版ソースコード。
テクスト生成アルゴリズム及び辞書データはHyperCardスタックからの移植です。
オリジナルのHyperCardスタックは移植者の手元から失われてるため、アイコン、背景画像はルーベンス「メデューサの首」の部分を使用しました。サウンドはありません。

テクスト生成アルゴリズム及び初期辞書データの著作権はAkiyoshi Sumi氏に属します。

このソースコードにはApache License 2.0が適用されます。

ネットワークの接続確認にHiroshi Hashiguchi氏のFBNetworkReachiablityを使用しました（ARC対応のため一部改変）。
FBNetworkReachiablityにはMIT Licenseが適用されます。

タブアイコンにはThe Working GroupのTWG Ratina Iconsを使用しました。
タブアイコン（CadavreExquis4iOS/images/*_24.png, *_24@2x.png）にはCreative Commons Attribution Share-Alike 3.0が適用されます。
